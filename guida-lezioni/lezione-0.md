# Prima della prima lezione

Benvenuti al corso **Informatica di Base**.

Prima di incontrarci, vi chiedo di fare tre cose. Non richiedono conoscenze pregresse: seguite i passi nell'ordine.

---

## 1. Installate Python

Andate su **python.org/downloads** e scaricate l'ultima versione stabile (3.12 o superiore).

Durante l'installazione su Windows: spuntate **"Add Python to PATH"** prima di cliccare Installa.

Per verificare che l'installazione sia andata a buon fine, aprite il terminale (su Mac: Terminale; su Windows: PowerShell o Prompt dei comandi) e scrivete:

```
python --version
```

Dovreste vedere qualcosa come `Python 3.12.x`. Se vedete un errore, non preoccupatevi: ne parliamo a lezione.

---

## 2. Installate Visual Studio Code

Scaricate **VS Code** da [code.visualstudio.com](https://code.visualstudio.com/download). È un editor gratuito e leggero, che useremo per scrivere ed eseguire il codice.

Dopo l'installazione, aprite VS Code e installate l'estensione **Python**:
- cliccate sull'icona delle estensioni nella barra laterale sinistra (o `Ctrl+Shift+X` / `Cmd+Shift+X`)
- cercate "Python" (quella di Microsoft)
- cliccate Installa

---

## 3. Familiarizzate con il terminale

Il terminale (o riga di comando) è uno strumento che useremo spesso. Non serve diventare esperti: basta saper fare queste tre cose.

**Sapere dove si è:**
```
pwd
```
Mostra la cartella in cui vi trovate.

**Muoversi tra cartelle:**
```
cd Documents
cd ..
```
`cd` seguito dal nome di una cartella ci sposta lì dentro. `cd ..` torna indietro di un livello.

**Vedere cosa c'è in una cartella:**
```
ls          # Mac/Linux
dir         # Windows
```

Provate a navigare fino alla vostra cartella Documenti e tornare indietro. Basta questo.

---

## Se qualcosa non funziona

Non bloccatevi. Segnate cosa è andato storto e ne parliamo all'inizio della prima lezione. L'installazione dell'ambiente è parte del corso.
